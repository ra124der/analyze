Welcome to the Data Processing Pipeline

This application demonstrates a robust data processing workflow. It converts an Excel spreadsheet (data.xlsx) into a CSV format (data.csv), processes the data using a Python script (execute.py), and publishes the results as a JSON file (result.json) via GitHub Pages.

The entire process is automated through a GitHub Actions workflow, ensuring code quality with Ruff and reliable data generation.

Visit GitHub RepositoryView Live Result JSON

Key Features:

  • Automated data conversion from Excel to CSV.
  • Python 3.11+ and Pandas 2.3+ compatible data processing.
  • GitHub Actions CI/CD for linting (Ruff) and execution.
  • Automated publication of result.json to GitHub Pages.
  • Clean, maintainable Python code (execute.py).
